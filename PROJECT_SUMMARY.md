## 🎯 How to Use

### 1. Edit Your Content (2 minutes)

**File:** `/config/site-config.js`

```javascript
student: {
  name: "Your Name",
  studentId: "20261234",
},
graduation: {
  date: "2026-06-15",
  time: "09:00",
  address: "Your Location",
  schoolName: "Your School",
}
```

### 2. Add Your Photo (1 minute)

Place your photo at: `/private-assets/graduation-photo.png`

### 3. Test Locally (1 minute)

```bash
python -m http.server 8000
# Open http://localhost:8000
```

### 4. Deploy Online (10 minutes)

Follow [DEPLOYMENT.md](DEPLOYMENT.md) to push to GitHub Pages
